## Markdown view

markdown文件浏览器实时预览

### Install
```bash
 npm install -g markdown-view
```

### Usage

```bash
 markdown-view [options] <filename>
```

### Options

```bash
   -V, --version          output the version number
   -o, --open <filename>  View markdown on your browser
   -h, --help             output usage information
```
